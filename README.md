# Drive-By-Power

A mod heavily based on Drive-By-Wire by Endalion and the Sable port by Rew1nd-dev, so lots of credit to them.

This adds wireless invisible power transfer "wire" networks, which support Sable sub-levels and can be used to power anything that supports FE, much like how Drive-By-Wire works with redstone signals.

They work between sub levels (so one ship can technically power another ship), and can transfer power from any FE source to any FE sink with a configurable server max power transfer rate.
